# foodKit
Full stack website for logging recipes and finding new recipes. Application utilizes the mealDB API

Tech stack:
- HTML
- Javascript/Jquery
- CSS/Boostrap
- Python
- Django

# Dependencies:
Package    Version
---------- -------
- bcrypt     3.2.0
- cffi       1.14.5
- Django     2.2
- pip        21.1.2
- pycparser  2.20
- pytz       2021.1
- setuptools 56.0.0
- six        1.16.0
- sqlparse   0.4.1
